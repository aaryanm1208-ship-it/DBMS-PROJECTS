# Library Management System

**Description:**  
Manage books, members, and borrowed books efficiently.

**Key Features:**  
- Add, update, delete books and members  
- Borrow and return books  
- Search books by author or title  
- Generate borrowed books reports

**Tables:**  
- Books (BookID, Title, Author, Publisher, Year, Copies)  
- Members (MemberID, Name, Email, Phone)  
- BorrowedBooks (BorrowID, MemberID, BookID, BorrowDate, ReturnDate)
