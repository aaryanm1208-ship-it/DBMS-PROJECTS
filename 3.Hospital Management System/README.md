# Hospital Management System

**Description:**  
Manages patients, doctors, appointments, and billing.

**Key Features:**  
- Add patient and doctor details  
- Schedule appointments  
- Manage billing  
- Generate appointment and bill reports

**Tables:**  
- Patients (PatientID, Name, Age, Gender, Contact)  
- Doctors (DoctorID, Name, Specialty, Contact)  
- Appointments (AppointmentID, PatientID, DoctorID, AppointmentDate, Diagnosis)  
- Billing (BillID, AppointmentID, Amount, Paid)
