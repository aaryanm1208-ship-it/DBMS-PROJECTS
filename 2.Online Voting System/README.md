# Online Voting System

**Description:**  
Secure online election system for voters and candidates.

**Key Features:**  
- Voter registration and authentication  
- Candidate management  
- Cast vote with validation  
- Real-time vote counting

**Tables:**  
- Voters (VoterID, Name, Email, HasVoted)  
- Candidates (CandidateID, Name, Party)  
- Votes (VoteID, VoterID, CandidateID, VoteDate)
